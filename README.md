# 📊 Sales EDA Dashboard (Excel)

## Project Overview
This project focuses on Exploratory Data Analysis (EDA) and Business Intelligence using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts

## Analysis
- Sales by Region
- Top 5 Products
- Sales by Category
- Sales Channel Analysis
- Customer Type Analysis
- Payment Method Analysis

## Dashboard
The dashboard includes:
- Total Revenue
- Total Orders
- Average Sales
- Total Quantity

## File
- Excel Dashboard (.xlsx)

## Conclusion
This project demonstrates how Excel can be used for data analysis and dashboard creation.
